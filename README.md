# JavaFSE Practice

This repository contains study material and practice code for Java Full Stack Engineering, organized by modules.

## Repository Structure

- `Upskilling/`
  - `Module 1/`
    - `Bootstrap5/` - Custom Bootstrap CSS and layout practice.
    - `CSS3/` - CSS style practice.
    - `HTML5/` - HTML practice files and example pages.
    - `JavaScript/` - Basic JavaScript practice.
  - `Module 2/`
    - `ANSI SQL Using MySQL/` - SQL exercises and setup scripts for MySQL.
  - `Module 3/`
    - `Core Java/` - Java practice programs covering fundamentals, OOP, exception handling, file I/O, collections, JDBC, concurrency, and more.

## Module Details

### Module 1 — Front-End Basics

This module includes practice files for:
- HTML5 structure and page examples.
- CSS3 styling techniques.
- Bootstrap 5 customization.
- JavaScript basics.

### Module 2 — SQL and Database Fundamentals

This module focuses on MySQL and ANSI SQL exercises.
- `Event_Management_Setup.sql` — likely schema and initial data setup.
- `Event_Management_Exercises.sql` — practice queries and SQL tasks.

### Module 3 — Core Java

This module contains Java source files for core concepts and exercises, including:
- Hello world and basic Java syntax.
- Control flow examples (loops, conditional checks, leap year, even/odd, grade calculator).
- Methods, recursion, arrays, string operations, and collections.
- Classes, objects, inheritance, interfaces, custom exceptions.
- File reading and writing.
- JDBC examples, transactions, and database operations.
- Threads, executor service, lambda expressions, Stream API, records, pattern matching, reflection, and virtual threads.

## How to Use

1. Open `Upskilling/` in your editor.
2. Explore each module folder and work through the code examples.
3. Run Java files in `Upskilling/Module 3/Core Java/` using your Java SDK.
4. Open HTML and CSS files in `Upskilling/Module 1/` with a browser to preview front-end exercises.
5. Execute SQL scripts in a MySQL environment for Module 2 practice.
